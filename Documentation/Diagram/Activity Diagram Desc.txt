	This Activity Diagram shows the activities that will happen in the system. The Quality Assurance Department will produce the QR code and the checklist. The Duty Mananger will assign a schedule to the Duty Inspector. The Duty Inspector will execute the inspection. After executing the inspection, the Duty Inspector will submit the result and time frame. The H2H Application will generate the result and time frame. The Duty Manager will check the generated result and time frame. If the result is incorrect, the Duty Manager will inform the Duty Inspector that the result is incorrect and the Duty Inspector needs to execute the inspection again. If the result is correct, the Duty Manager will generate a final report. If there are no issues detected, the final report will be stored automatically. If there are issues detected, the Quality Assurance department will address the issues. After addressing the issues, the report will be stored.
