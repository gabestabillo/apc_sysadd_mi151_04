This Activity Diagram shows the Execute Inspection process. The actor that is involved in this process is the Duty Inspector. The Execute Inspection process will start on what scheduled facility is selected. The Duty Inspector will Scan in the QR code and the time frame will start up. The Duty Inspector will inspect the facility using the generated electronic checklist and set on what is the status of the facility. Finally, the duty inspector will scan out the QR code and the time frame will end up.
